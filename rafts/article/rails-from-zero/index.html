Rails from Zero

Ruby 2.1.3 | Rails 3.2.1 | Jan 14, 2015

Prerequisites

  • RVM or rbenv
  • Bundler

Create a new Rails app

rails new <app_name> -T --database=postgresql --skip-turbolinks

This gives us a new Rails app with no test framwork (-T), a postgrsql database(--database=postgresql), and no turbolinks (--skip-turbolinks).

Add Gems:

  gem 'pry-rails'

group :test, :development do
  gem 'rspec-rails'
  gem 'capybara'
  gem 'launchy'
  gem 'database_cleaner', '~> 1.4.1'
end

If Minitest is your thing, swap out the rspec spsific gem with:

group :test, :development do
  gem 'minitest-rails'
  gem 'guard-minitest', require: false
end

With test frame work chosen and gems in place run:

bin/bundle

Initialize & Configure Testing Framework

Next depending on the test-framework run the installation generator:

Rspec

rails generate rspec:install

This adds the following files which are used for configuration:

- .rspec
- - spec/spec_helper.rb
- - spec/rails_helper.rb

Configure

Capybara

Setup

Add gems: ```ruby

Gemfile

group :test, :development do gem 'capybara' gem 'launchy' end ```

Configure: ```ruby

rails_helper.rb

require 'capybara/rails' ```

Database Cleaner

Create file and with: ```ruby

spec/support/database_cleaner.rb

require 'database_cleaner'

RSpec.configure do |config|

config.before(:suite) do DatabaseCleaner.strategy = :transaction DatabaseCleaner.clean_with(:truncation) end

config.around(:each) do |example| DatabaseCleaner.cleaning do example.run end end

end ```

Minitest

rails generate minitest:install

This will add the test_helper.rb file to the test/ directory.

Guard

Gard audimates test runs.

Add Gems:

group :test, :development do
  gem 'growl'
  gem 'guard'
  gem 'guard-rspec', require: false
end

guard init rspec __or__ minitest
